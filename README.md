# Bash Script Template

A minimal, REUSE-compliant template for Bash scripting projects.

## Features

- SPDX license headers
- GPL-3.0-or-later
- REUSE compliance
- ShellCheck via GitHub Actions
- Strict Bash defaults

## Usage

Copy `scripts/example.sh` and modify as needed.

```bash
./scripts/example.sh
